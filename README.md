# Mini Messageboard

A simple message board built with Node.js, Express, and EJS. Users can post messages and view them on the main page. Each message has a detail page showing the full post.

## Features

- Post messages with a name and text
- Messages display in reverse chronological order
- Individual detail page for each message
- Server-side rendering with EJS templates

## Tech Stack

- Node.js
- Express
- EJS

## Notes

Messages are stored in memory and do not persist between server restarts. This is intentional.
